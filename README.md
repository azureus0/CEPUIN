# Cara Menjalankan Aplikasi

1. **Ekstrak File ZIP**  
   Sebelum menjalankan, pastikan kamu sudah mengekstrak seluruh isi file ZIP ke dalam satu folder. Jangan jalankan langsung dari file ZIP.

2. **Pastikan Streamlit Terinstal**  
   Aplikasi ini membutuhkan Streamlit agar bisa dijalankan. Jika belum terinstal, jalankan perintah berikut di terminal:
   pip install streamlit

3. **Jalankan Aplikasi**  
   Setelah Streamlit terinstal, masuk ke folder hasil ekstraksi, lalu jalankan perintah berikut di terminal:
   streamlit run app.py

4. **Login ke Aplikasi**  
   Setelah dijalankan, aplikasi akan terbuka otomatis di browser. Gunakan salah satu akun berikut untuk login:
   - Sebagai User:  
     Username: wadaw  
     Password: 123  
   - Sebagai Admin:  
     Username: admin  
     Password: admin123
